# AtlassianConfluenceScraping
Scrapes Atlassian confluence to migrate to another platform. No admin access required only user access.
This script set was created to start the scrape of confluence without an admin account. Give it to a juniour worker to run with, migrate documents and then put onto another wiki, or just for backups/reformatting as the team moves on.

Run confluence auth test python first (make sure you have python on your machine first)
This will check that you have the access to start scraping your site for download.

This is for when you don't have access to admin the page as admins have left or licence's are limited
